# Bundled fonts

These font files are self-hosted (rather than loaded from Google Fonts) so the
tool makes no third-party request when a visitor opens it — a GDPR-friendly
choice, and one that keeps the tool self-contained.

Both families are free and open source, released under the
**SIL Open Font License 1.1** (https://scripts.sil.org/OFL):

- **Fraunces** — © The Fraunces Project Authors
  (https://github.com/undercasetype/Fraunces)
- **Inter** — © The Inter Project Authors
  (https://github.com/rsms/inter)

Only the `latin` and `latin-ext` subsets are included, in the weights the
interface uses.
